# naming-book
- 추가 :
add, insert, 
- 수정 :
modify, update,
- 삭제 :
remove, delete
- 동기 :
sync, refresh, reload
- 유효성 검사 :
check ( 반환값 void ), validate
- 실행 :
do, execute, process, fire, request
- 찾기 :
find, search
- 추출 :
extract, get
- 채우기 :
fill, inject, set, input
- 포함 :
contain, has, is
- 변환 :
map, convert
- 조작 : 
toggle, handle
- 불러오다 :
load
- 이동 :
move, navigate, shift
- 리스너 :
on
- etc
